---

Помоги мне спланировать и реализовать новую функцию в проекте.

**Шаги:**

1. **Анализ требований:**
   - Что именно нужно реализовать?
   - Какие есть edge cases?
   - Как это вписывается в существующую архитектуру?

2. **Планирование:**
   - Какие модули нужно создать/изменить?
   - CLI команды (cli.ts)?
   - API endpoints (server.ts)?
   - Новые функции экспорта (exporter.ts)?

3. **Безопасность:**
   - Какие риски? (см. SECURITY.md)
   - Сохраняется ли privacy by default?
   - Нет ли новых внешних API вызовов?

4. **Разбивка на задачи:**
   - Разбей реализацию на небольшие шаги
   - Определи порядок выполнения
   - Добавь в BACKLOG.md

5. **Реализация:**
   - Следуй паттернам из ARCHITECTURE.md
   - Используй существующие модули
   - Добавь типы TypeScript

6. **Тестирование:**
   - Проверь `npm run build`
   - Ручное тестирование
   - Проверь edge cases

**Принципы:**
- Следуй архитектуре проекта (ARCHITECTURE.md)
- Приватность по умолчанию
- Минимум новых зависимостей
- Код должен быть понятным

**После реализации:**
- Обновить BACKLOG.md
- Обновить PROJECT_SNAPSHOT.md
- Добавить в CHANGELOG.md
