Помоги протестировать указанный функционал.

**Типы тестирования для claude-export:**

1. **CLI тестирование:**
   - `npm run dialog:list` — список сессий
   - `npm run dialog:export` — экспорт
   - `npm run dialog:ui` — веб-интерфейс
   - `npm run dialog:watch` — watch режим

2. **Экспорт:**
   - Новые сессии экспортируются корректно
   - Markdown форматирование правильное
   - Метаданные заполнены
   - Author attribution работает

3. **Visibility:**
   - По умолчанию в .gitignore
   - Toggle работает в UI
   - .gitignore обновляется корректно

4. **Watch режим:**
   - Новые файлы обнаруживаются
   - Debounce работает
   - Summary генерируется

5. **Edge cases:**
   - Пустые сессии
   - Большие сессии (1000+ сообщений)
   - Специальные символы в сообщениях
   - Несуществующие пути

**Чеклист тестирования:**

- [ ] `npm run build` проходит без ошибок
- [ ] CLI команды работают
- [ ] UI открывается и отображает диалоги
- [ ] Toggle visibility работает
- [ ] Новые экспорты в .gitignore
- [ ] Watch обнаруживает изменения

**После тестирования:**
Сообщи о найденных проблемах для добавления в BACKLOG.md
