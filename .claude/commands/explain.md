description: Объяснить как работает код или функционал
---

Объясни как работает указанный код или функционал.

**Что объяснить:**

1. **Высокоуровневый обзор:**
   - Для чего этот код?
   - Какую проблему решает?
   - Как вписывается в архитектуру?

2. **Детали реализации:**
   - Какие функции/классы используются?
   - Как происходит data flow?
   - Какие зависимости?

3. **Ключевые решения:**
   - Почему так, а не иначе?
   - Какие альтернативы рассматривались?
   - Какие trade-offs?

**Модули claude-export:**

- `cli.ts` — Entry point, парсинг команд
- `exporter.ts` — JSONL → Markdown конвертация
- `server.ts` — Express сервер и API
- `watcher.ts` — Chokidar file watching
- `gitignore.ts` — .gitignore manipulation

**Формат ответа:**

1. Краткое резюме (1-2 предложения)
2. Детальное объяснение с примерами кода
3. Диаграмма data flow если уместно
4. Связь с другими частями системы
